# JetLag CZE

This is an auxiliary repository containing needed tooling for playing
JetLag in Czechia.

Toto je pomocný repozitář pro nástroje potřebné k příjemnému hraní
JetLag her v Česku.

## Rules

It is OK to vibecode in this repository as long as the person owning the
clanker actually does code reviews. Send a pull request for any changes,
open an issue if anything looks off.
